## Phishing Awareness Project using Gophish

During my employment, I was assigned to a project to simulate phishing attacks within the company. The goal was to check how many employees clicked the links and who reported the phishing incidents.

## Introduction

Cyberattacks are one of the biggest threats in today’s digital world. Attackers use techniques like:
- Malware
- Phishing
- Ransomware
- Social engineering

Among these, phishing is one of the most common.
Phishing attacks trick users into giving away sensitive information (like usernames, passwords, or financial details) by pretending to be a trusted source.

This project uses Gophish, a phishing simulation tool, to help organizations:
- Educate employees
- Simulate real phishing attacks
- Improve overall cybersecurity awareness

## Purpose

The goal of this project is to reduce the risk of phishing attacks within an organization by focusing on human behavior.
This includes:
- Raising awareness about phishing tactics
- Testing how users respond to phishing emails
- Training users to make safer decisions online
- Preventing data breaches and cyber incidents

## Objectives
1. Identify vulnerable users
Run simulated phishing campaigns to see who:
- Clicks malicious links
- Submits sensitive information
2. Improve cybersecurity awareness
Educate employees about:
- Phishing attacks
- Social engineering
- Malicious attachments
3. Reduce cyber risk
Help users:
- Recognize suspicious emails
- Avoid unsafe actions
4. Evaluate training effectiveness
Measure:
- User responses
- Improvement over time
5. Strengthen security posture
Better user awareness = fewer successful attacks
