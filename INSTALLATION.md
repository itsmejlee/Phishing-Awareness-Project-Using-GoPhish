## Installation
Step 1: Download Gophish
- Go to: https://github.com/gophish/gophish/releases
- In this case, I hosted the project in Windows 11.

**Note**: *Download the version for your operating system.*

Step 2: Extract files
- Extract the ZIP file
- Move it to your preferred location

**Example**: C:\Users\YourName\Documents\gophish

Step 3: Run Gophish
- Open the folder
- Run: gophish.exe

Step 4: Access the Web Interface

After running, you’ll see:
- **Admin Server:** Local IP: https://127.0.0.1:3333 (Used for Admin dashboard and configuration)
- **Phishing Server:** http://0.0.0.0:80 (Used for capturing data or this where your Landin Pages hosted)

**Note**: *Save the default username and password shown in the terminal.*

For more information regarding installation such as you want to access the Admin Panel to the internet or manipulating the config.json. <br>
Go to: https://docs.getgophish.com/user-guide/installation
